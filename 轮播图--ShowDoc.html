轮播图

简要描述：

  • 轮播图

请求URL：

  • https://api-hmugo-web.itheima.net/api/public/v1/home/swiperdata

请求方式：

  • GET

参数： 无

返回示例

  1. {
  2.     "message": [
  3.         {
  4.             "image_src": "https://api-hmugo-web.itheima.net/pyg/banner1.png",
  5.             "open_type": "navigate",
  6.             "goods_id": 129,
  7.             "navigator_url": "/pages/goods_detail/index?goods_id=129"
  8.         }
  9.     ],
  10.     "meta": {
  11.         "msg": "获取成功",
  12.         "status": 200
  13.     }
  14. }

返回参数说明

参数名 类型 说明
image_src string 图片路径
open_type string 打开方式
goods_id number 商品id
navigator_url string 导航链接

备注

  • 更多返回错误代码请看首页的错误代码描述
